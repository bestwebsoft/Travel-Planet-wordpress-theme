== Description ==

Are you an traveler who looks for a perfect theme that would highlight your stories without being too distracting? Or are you a handyman who wishes to share the nowledge with people worldwide via simplistic yet utterly elegant blog? Or maybe you seek a theme with an exquisite and balanced interface for a travel or photography magazine? Whether your purposes match or transcend our guesses, we are delighted to introduce Travel Planet - a streamlined and professional theme that will serve as a perfect entourage for your ideas. Travel Planet is ideal for Tourist Agencies, Travel Diaries, Tourist Destinations, Travel Magazines, Travel Guides, Photography, Handmade and DIY Workshops, Cuisine, Event Information, Culture Overviews and Tutorials, or any other purposes you feel are appropriate.

== Installation ==

Manual installation:

1. Upload the 'travel-planet' folder to the '/wp-content/themes/' directory

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'travel-planet'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activation and Use

1. Activate the Theme through the 'Themes' menu in WordPress

== Theme Notes == 

= Slider =

To activate the slider, you need to select posts, which should be seen in the slider. Without these posts the slider will not be displayed. To add a post or a page into the slider, go to admin panel (wp-admin), and click edit post (page). Upload the slider image: "Featured image" -> "Set featured image". Pay attention: for correct work of the slider the needed image width should be not less 940 px and height not less than 315 px. 
In "Slider" field put a mark on "Do you wish to add this post to slider?". Add a comment for the slide post ("Caption for post in slider"). Save the post ("Publish"). Congratulations, you just add a new slide into the slider. 
If you have one single slide, the slider will be displayed as a static image. 
You can have as much slides as you want.  
Slider is located on the main page and consists of three parts: the title (post title), a comment (comment for the slide post) and image ("Featured image");

= Post Thumbnail Functionality =

For post thumbnails we recommend to use the image width should be not less then 560 px., if the post was not added into the slider п. In other case the image size should be higher then  940 px. width and 315 px. height.

= Post Formats =

Travel Planet Wordpress do not support formats. All posts will de displayed as regular messages.

== Frequently Asked Questions ==

= How can I change language from English into Russian? =

In the main catalog of your WordPress site find file wp-config.php, open it and change define( ‘WPLANG’, '' ); for define( ‘WPLANG’, ‘ru_RU’ );

= Does the theme support widgets? =

Yes, this theme includes widgets. The following widgets are shown by default: search, recent posts, recent comments, archives, categories.

= How to upload background image into the title? = 

You can use background image for the title of your website. Use the menu to select Header -> Choose Image -> Browse, then click Upload. Background title images should be 1920px * 160px.
